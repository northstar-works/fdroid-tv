Get Started

Add this repo to Flikky / F-Droid

https://northstar-works.github.io/fdroid-tv/repo
1 Install Flikky from the Amazon Appstore on your Fire TV
2 Open Flikky → tap Repositories → Add Repository
3 Enter the repository URL above and confirm
4 Refresh — StreamVault TV will appear ready to install
Scan to add on your device Point your phone camera at this code
to open the repo URL directly.
